Restaurant QR-Based Ordering System

Overview


A full-stack web application that enables restaurants to manage orders through table-specific QR codes. Each table is assigned a unique QR code which, when scanned by customers, displays the restaurantâ€™s live menu and allows them to place orders. Restaurant staff can manage tables, automatically generate QR codes, view incoming orders, and update order statuses (Pending, Preparing, Completed) through an admin dashboard. The system streamlines order management and eliminates the need for physical menus.
